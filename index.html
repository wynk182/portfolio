"Cork"

Cork is a virtual bulletin board web app built with ruby on rails. Cork incorporates a blog style posting and commenting mechanism on a fun cork board styled platform. Users can create content anonymously or after creating an account. All posts are verified by board owners or SA depending on permissions before displayed. All boards start un-owned until they are claimed and google embeding api is used to display a local location for boards with addresses. Users can recieve replys to their post via the web application, email, or text messaging based on their contact information.
