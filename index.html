WORK

"Mossbee"

A mobile ordering solution for small businesses built with ruby on rails. Pay as you go business model with text to order functionality. Vendors upload menus/items from a square account and their customers can view the menu by sending a text to the corresponding phone number. Text messaging is handled through Twilio api and payment processing is handled through Square api.

"Cork"

A virtual bulletin board web app built with ruby on rails. Cork incorporates a blog style posting and commenting mechanism on a fun cork board styled platform. Users can create content anonymously or after creating an account. All posts are verified by board owners or SA depending on permissions before displayed. All boards start un-owned until they are claimed and google embeding api is used to display a local location for boards with addresses. Users can recieve replies to their post via the web application, email, or text messaging based on their contact information.
